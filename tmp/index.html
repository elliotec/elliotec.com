Hi, I'm Michael Elliott.


I'm a full-stack web developer known professionally and around the web as elliotec. I'm currently available for contract/consulting work or a full-time position.

About Me

Development

I use Ruby on Rails as my primary development framework for creating dynamic web applications.

Design

I use HTML5 & SCSS for design and layout, and jQuery is a go-to for moving stuff around the page.

Deployment

Heroku makes deployment super easy, but I also use AWS, Digital Ocean, and Site44 for production. My favorite DNS provider is IWantMyName.

Integration

I have experience with iOS and hardware integration, building APIs, and consuming APIs.

Tools

I use OSX and Linux for development, Git for version-controll, and spend lots of time on the command line.

Hobbies

I enjoy rock climbing, snowboarding, video games, ping-pong, friends, playing music, travelling, and a bunch of other stuff too.

What can I do for you?

Have an app idea? Need an info, ecommerce, portfolio site, or a full-time developer? I can rapidly prototype web applications with the whole package or help on just about any project.

Some Projects

Beautilist

Beautilist is a Rails app that implements a tree data structure in Ruby to infinitely nest lists and uses Ajax calls to manipulate the lists and their children around the DOM in real-time.

Sign up to add as many lists and sublists as you want to see what the app is capable of.

Makaela Herran Fine Art Photography

This was one of my first projects, and I will always love it for how much I learned from it. It was my first exposure to jQuery, and designing the entire thing from scratch taught me a ton about HTML and CSS.

Also, Makaela is a great Photographer. Hire her if you need some photos!

Fastbar

Redesign Seattle Startup Weekend - 3rd Place Winner

FastBar is an event process improvement company combining iOS, Rails, and RFID technologies.

I engineered the back-end, from the registration system to the data analytics. I worked closely with the iOS developer integrating the native POS application with the web app and database, and worked with the designers on making it all look great.

Most of the action happens on the iOS side, but check out the code on my Github to see how it all works.

Code Union

Code Union is an application aggregating and curating resources and best practices for people learning to code. I led a team of 4 in the development and design of the initial prototype.

The application has since been taken over by a different team and is still in development for a couple weeks. As such, the site may be down periodically for matinence or feature changes, and content will be sporadically added and removed.

In the mean time, feel free to sign up and add your own content to see how it works!
